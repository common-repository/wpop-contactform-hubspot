Add Contact Form 7 Data to Hubspot Contact lists.

== Description ==

Contact Form 7 is the most popular WordPress contact form builder and HubSpot & Contact Form 7 developed as addon of Contact Form 7, helps to extend your subscription lists and collect unlimited leads on HubSpot.You can easily Contact Form 7 data to HubSpot Contact List.



=== Features === 

Adds subscription in "HubSpot" through "Contact Form 7"
Option to select "HubSpot" list ID for subscribers
Option to select "Contact Form 7" fields for "HubSpot" list
Constant updates

== Installation ==
1. Unzip wpop-contactform-hubspot.zip
2. Upload all the files to the /wp-content/plugins/wpop-contactform-hubspot
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. For customizing the plugin's settings, open "Contact Form 7"  form and go to "Hubspot" tab
